= JBANG-WRAPPER-INSTALL

== NAME

jbang wrapper install -- Install/Setup jbang as a wrapper script in a folder

== SYNOPSIS

[source]
----
jbang wrapper install [-hfox] [-d=<dir>] [--config=<config>] [--insecure] [--verbose] [--quiet] [--fresh]
----

== DESCRIPTION

Install/Setup jbang as a wrapper script in a folder

== OPTIONS

*-h*, *--help*::
Display this help and exit

*-d*, *--dir*=<dir>::
The folder to install the wrapper into.

*-f*, *--force*::
Force installation of wrapper even if files already exist

*--config*=<config>::
Path to config file to be used instead of the default

*--insecure*::
Enable insecure trust of all SSL certificates.

*--verbose*::
jbang will be verbose on what it does.

*--quiet*::
jbang will be quiet, only print when error occurs.

*-o*, *--offline*::
Work offline. Fail-fast if dependencies are missing.

*--fresh*::
Make sure we use fresh (i.e. non-cached) resources.

*-x*, *--stacktrace*::
Print exceptions stacktraces to stderr (even when quiet).
