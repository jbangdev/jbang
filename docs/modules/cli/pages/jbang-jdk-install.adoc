= JBANG-JDK-INSTALL

== NAME

jbang jdk install -- Installs a JDK.

== SYNOPSIS

[source]
----
jbang jdk install [-hfox] [--config=<config>] [--insecure] [--verbose] [--quiet] [--fresh] <versionOrId> [<existingJdkPath>]
----

== DESCRIPTION

Installs a JDK.

== OPTIONS

*-h*, *--help*::
Display this help and exit

*-f*, *--force*::
Force installation even when already installed

*--config*=<config>::
Path to config file to be used instead of the default

*--insecure*::
Enable insecure trust of all SSL certificates.

*--verbose*::
jbang will be verbose on what it does.

*--quiet*::
jbang will be quiet, only print when error occurs.

*-o*, *--offline*::
Work offline. Fail-fast if dependencies are missing.

*--fresh*::
Make sure we use fresh (i.e. non-cached) resources.

*-x*, *--stacktrace*::
Print exceptions stacktraces to stderr (even when quiet).

== ARGUMENTS

*<versionOrId>*::
The version or id to install

*<existingJdkPath>*::
Pre installed JDK path
