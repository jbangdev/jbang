= JBANG-APP-SETUP

== NAME

jbang app setup -- Make jbang commands available for the user.

== SYNOPSIS

[source]
----
jbang app setup [-h] [--[no-]java] [--force] [--config=<config>] [--insecure] [--[no-]verbose] [--[no-]quiet] [--[no-]offline] [--[no-]fresh] [--[no-]stacktrace]
----

== OPTIONS

*-h*, *--help*::
Display this help and exit

*--[no-]java*::
Add JBang's Java to the user's environment as well

*--force*::
Force setup to be performed even when existing configuration has been detected

*--config*=<config>::
Path to config file to be used instead of the default

*--insecure*::
Enable insecure trust of all SSL certificates.

*--[no-]verbose*::
jbang will be verbose on what it does.

*--[no-]quiet*::
jbang will be quiet, only print when error occurs.

*-o*, *--[no-]offline*::
Work offline. Fail-fast if dependencies are missing.

*--[no-]fresh*::
Make sure we use fresh (i.e. non-cached) resources.

*-x*, *--[no-]stacktrace*::
Print exceptions stacktraces to stderr (even when quiet).
